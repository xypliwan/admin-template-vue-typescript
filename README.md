# vue + typescript + vuex + element UI + scss  后台基础模板

## 简介
使用vue-cli3.0 搭建的后台基础模板。 vuex全局状态管理。

## 项目全局功能
1：登录  
<br />
2：全局的axios 请求配置
<br />
3: 动态配置菜单栏，支持多级
<br />
4: 全局路由拦截
<br />
5: 页面 动态keep-alive
<br />
6: 页面 跳转动画
<br />
7: 多环境接口配置、打包 （ 本地环境、测试环境、线上环境 ）


## Project setup
```
npm install
```

### 本地
```
npm run serve
```

### 线上打包
```
npm run build
```

### 测试打包
```
npm run test
```

### 本地打包
```
npm run intr
```
